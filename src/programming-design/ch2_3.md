# Chapter 2.3 static

### 1.static

```c
void function(){
  static int temp = 0;
  temp++;
}
```

> **Note** <br />
> - static一開始初始化是0，但在每次呼叫function後static值變不再初始化
> - [static pointer](https://iq.opengenus.org/static-and-extern-pointers-in-c/)有點進階，但還是可以看看
