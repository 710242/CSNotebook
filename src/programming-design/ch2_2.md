# Chapter 2.2 const

### 1.define

```c
#define A 10
#define F(a,b) (a) * (b)
```

> **Note** <br />
> - 雖然可能不會考，但第二個例子要加括號，ex：a = 5 + 9 ; b = 1 + 7 ; f(a,b) = 21
> - define變數名稱全部大寫（大家都這麼做）

### 2.const

```c
int main(){
  const a = 10;
}
```

> **Note** <br />
> - const雖然不會變，但有[黑魔法](https://stackoverflow.com/questions/12104839/how-to-modify-value-of-const-variable/31583505)
