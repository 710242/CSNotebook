# Chapter 3.1 while

### 1.while

```c
int main(){
  while (condition is true) {
    do something
  }
}
```

### 注意
- while迴圈都只做"不知道次數"的事情（不用我說again）
- while 其實就是 for(; ending condition ;)，不用特別記

### 2.do while

```c
int main(){
  do{
    do something
  }while(condition is true);
}
```
### 注意
- do while一定會先做一次，不過通常還是用while比較多
