### 1.printf

```c
int main(){
  printf("%d %c %s %f\n", int, char, char[], float);

  printf("%o %x %u\n", 8, 16, 10);

  printf("%100d %-100d %.9f\n");

  printf("%ld %sd\n");
}
```

> **Note** <br />
> - %s會自動讀到字串陣列結尾（可記可不記？）
> - o x u 分別表示8進位 16進位 10進位（unsigned）
> - 型態前面加數字是留n個空格，小數點後則是輸出到小數點第n位
> - l 即是 long , s 即是 short
